--- wordpress_id: 143 title: Flickrpress 1.0 wordpress_url: http://spaetzel.com/PermaLink,guid,228.aspx layout: post --- I have created a php script entitled Flickrpress. This script will get all of the photos uploaded to flickr by a specific user in the last 24 hours. It will then post the thumbnails to a Wordpress blog. You can see the results here.


To use the script, just download the source and place it on your web server. Then set up the script file to be run daily. The full instructions are located in the source file.
