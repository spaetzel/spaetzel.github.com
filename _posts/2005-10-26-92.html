Well I have been giving a new web browser a whirl over the last day. The brower's name is Flock and it is looking pretty cool. It is called a "Social Browser' and it has many features that lend itself to what is being called Web 2.0. These features include del.icio.us, Flickr and weblogging integration.

Thus far the browser does seem a bit buggy. I had trouble getting it to post to my Drupal based weblog. But it looks like the developers are very helpful and answered my question on their forums almost immediately.

It definitely shows promise. Web users who are intersested in the latest Social web technologies should definitely check it out.
