--- wordpress_id: 49 title: Flickrpress wordpress_url: http://spaetzel.com/PermaLink,guid,484.aspx layout: post --- I have created a php script entitled Flickrpress. This script will get all of the photos uploaded to flickr by a specific user in the last 24 hours. It will then post the thumbnails to a Wordpress blog.

To use the script, just download the source and place it on your web server. Then set up the script file to be run daily. The full instructions are located in the source file.

Please add any questions or suggestions to the comments. Any feedback would be appreciated!

Update: Flickrpress has been tested with Wordpress 1.2 and Wordpress 1.5
